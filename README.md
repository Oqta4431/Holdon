## サービス概要
衝動買いを抑制するために「立ち止まって考える」習慣を支援する自己管理アプリです。
欲しいと思った瞬間に登録し、一定期間後に再考することで購買行動を見直すことができます。
節約額ではなく、熟考した回数や行動の変化を可視化します。

## このサービスへの思い・作りたい理由
普段の何気ない衝動買いが積み重なって、後悔することが多くありました。
家に代用品があったり、別の店でもっと安く売っていたり、買った直後に熱が冷めて使わなくなったり、買っても使う時間がなかったりと後悔の理由はさまざまです。
これらの問題は本質的には、「その瞬間に立ち止まって考える時間を取らなかった」ことが原因だと気づきました。
このアプリは、そんな“勢いの買い物”の前に「一呼吸おく」きっかけを作るためのものです。
購入を我慢するためではなく、「今の自分にとって本当に必要か」を見つめ直すことで、より納得感のある選択ができるようにします。
一度立ち止まって考えたうえで選んだ買い物なら、たとえ買っても買わなくても、その後の満足度や自信につながるはずです。
お金の節約や反省ではなく、「自分の購買行動を意識的に選択できるようにする」ことがこのサービスの目的です。
衝動買いを「失敗」ではなく「学び」に変え、買い物を通じて自分をより深く理解する。
そんな体験を日常の中に届けたいと思い、このアプリを作ることを決意しました。

## ユーザー層について
- 衝動買いを抑えたい20〜30代の社会人・学生
- 節約志向・ミニマリスト傾向・自己管理に興味のある人

## サービスの利用イメージ
ユーザーは「欲しい」と思った瞬間にアプリを開き、商品を登録します。
24時間後に「まだ欲しい？」というメールが届き、再度考える機会が得られます。
その結果「購入する」「見送る」「再検討する」を選択でき、購買行動を振り返ることができます。
立ち止まる習慣が身につき、無駄な支出を減らしながら意思決定力を育てられます。

## ユーザーの獲得について
Xでのシェア機能で拡散し、その投稿を見たユーザの流入を想定しています。
SNSでの発信やミニマリスト・倹約家コミュニティでの共有を想定しています。
また、日々の生活改善をテーマとするブログやノート記事などからの流入も検討しています。

## サービスの差別化ポイント・推しポイント
同じような目的のアプリとして以下のようなものが挙げられます。
- [購入判断サポート-満足度の高い使い方でお金の価値を引き出す](https://apps.apple.com/us/app/%E8%B3%BC%E5%85%A5%E5%88%A4%E6%96%AD%E3%82%B5%E3%83%9D%E3%83%BC%E3%83%88-%E6%BA%80%E8%B6%B3%E5%BA%A6%E3%81%AE%E9%AB%98%E3%81%84%E4%BD%BF%E3%81%84%E6%96%B9%E3%81%A7%E3%81%8A%E9%87%91%E3%81%AE%E4%BE%A1%E5%80%A4%E3%82%92%E5%BC%95%E3%81%8D%E5%87%BA%E3%81%99/id6753330307)
- [Shopping Detox - 買い物依存症克服アプリ](https://apps.apple.com/jp/app/shopping-detox-%E8%B2%B7%E3%81%84%E7%89%A9%E4%BE%9D%E5%AD%98%E7%97%87%E5%85%8B%E6%9C%8D%E3%82%A2%E3%83%97%E3%83%AA/id6740816591)
- [いったん、やめとこ-節約記録と欲しいものリスト](https://apps.apple.com/us/app/%E3%81%84%E3%81%A3%E3%81%9F%E3%82%93-%E3%82%84%E3%82%81%E3%81%A8%E3%81%93-%E7%AF%80%E7%B4%84%E8%A8%98%E9%8C%B2%E3%81%A8%E6%AC%B2%E3%81%97%E3%81%84%E3%82%82%E3%81%AE%E3%83%AA%E3%82%B9%E3%83%88/id6746414009)
これらのアプリと比較したHoldonの特徴的な機能は「リマインド機能」です。
一時的な衝動を抑えるのに最も重要なことは「時間をおいて冷静に考えること」です。
これらの購入判断アプリは商品を登録したその場で購入の判断を迫られますが、これでは購入したい気持ちがまだ強く残っている状態のため選択肢を深く吟味できません。
Holdonでは購入の判断を遅らせることによって時間を取り、一度冷静になったうえで購入の判断を行うことが出来ます。
また、複数ECサイトや実店舗を横断した欲しいものリストを一元管理できるため、公開しないお金の使い道の思考をサポートします。

## 機能候補
【MVPリリース時】
- ユーザー登録（メール＋パスワード）
- 欲しいもの登録（タイトル／価格／リマインド期間）
- 「購入決定／見送り／再検討」判断機能
- ホーム画面（今月の熟考完了数の表示）
- リスト画面（登録中／検討完了／購入済みのタブ切替）

【本リリースまでに実装したい機能】
- メール通知（登録から24時間後）
- LINE通知機能（LINE Messaging API）
- URL自動抽出による商品情報取得
	- AmazonAPI（PA-API）
	- 楽天API(Rakuten Ichiba Item Search API)
- カレンダービューでの振り返り
- 週次レポートメール
- 満足度レビュー
	- 買う/買わないにかかわらず、判断の1週間後に自分の判断のレビューをする
	- 星の数とコメントでレビュー
- 「買いたい理由 vs 買わない理由」メモ欄
	- 2カラムのメモ欄
	- 買う理由VS買わない理由
